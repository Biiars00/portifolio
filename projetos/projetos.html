Setup School

O Setup School é uma plataforma para cadastro de matrícula escolar, acompanhamento de alunos matriculados por cursos/turmas, edição e exclusão de cadastro. Construído em Python e criação de API em Django.
TM Calculator

TM Calculator é um projeto baseado no Teorema de Pitágoras que permite calcular os lados dos catetos de um triângulo retângulo para encontrar o valor da hipotenusa.
Biiar Bank

Formulário de criação de contas para o banco virtual BiiarBank, com opção de solicitação de cartão de crédito ou débito para pessoa física ou jurídica.

Soup Zeer

O Soup Zeer é um restaurante que oferece variados tipos de sopas e tem como foco principal de entrega, o delivery. Este projeto fictício refere-se a página de contato da empresa que contém formulário, mapa de localização do estabelecimento e redes sociais linkadas.
